* Setup
** Virtualenv
If poetry is not installed:
#+BEGIN_SRC sh
brew install poetry
#+END_SRC
and restart your shell.

Install dependencies in virtualenv:
#+BEGIN_SRC sh
poetry install
#+END_SRC

To add dependencies:
#+BEGIN_SRC sh
poetry add package-name
#+END_SRC
